# Welcome to Water Jug FrontEnd!

# Installation

Run `npm i` to install dependencies

## Development

```sh
npm run dev
```

This starts your app in development mode, rebuilding assets on file changes. By default the app will be running on: `http://localhost:3000/`

### Folder structure
```
ui
└─── app
│   └─── components
│   └─── routes
│       │   root.tsx
│       │   ...
└─── test
└─── types
└─── utils
│   README.md (this file)
│   ...
```

npm run test
